## v1.1.0

- Added light interruptions after hull impacts. / Добавили перебои света после ударов об корпус.
- Added credits in the main menu. / Добавили титры в главное меню.
- Updated the emitter so it no longer breaks. / Обновили эмиттер, чтобы он больше не ломался.
- Added mobile hints for the emitter interaction. / Добавили подсказки для взаимодействия с эмиттером на мобильных устройствах.
- Fixed Android fullscreen without a top safe-area bar. / Исправили полноэкранный режим на Android без верхней safe-area полосы.
- Fixed enemy attack timing bugs. / Исправили ошибки в таймингах атак врагов.
- Updated difficulty progression. / Обновили прогрессию сложности.
- Lowered the game pace. / Темп прохождения более размеренный.
- Added clearer health warning feedback. / Добавили более понятную обратную связь по предупреждению о здоровье.
- Added game over when every submarine system is broken. / Новое условие поражения.
- Enemies lag behind earlier when surfacing. / Враги раньше отстают при всплытии на поверхность.
- Increased sonar ping volume. / Увеличена громкость импульса сонара.
- Added edge glow on the depth gauge to emphasize ascent and descent engine activity. / Добавлено свечение по краям шкалы глубины, чтобы подчеркнуть работу двигателей при всплытии и погружении.
- Fixed interactable hover feedback so availability visuals update while the cursor stays on the object. / Исправлена подсветка интерактивных объектов: визуальная доступность теперь обновляется, пока курсор остается на объекте.
- Added a dithering effect. / Добавлен эффект дизеринга.
- Fixed the CRT effect toggle so lens distortion is restored when CRT is turned back on. / Исправлено переключение CRT-эффекта: искажение линзы снова восстанавливается при повторном включении CRT.
- Linked chromatic aberration to the CRT effect toggle so it turns off together with the CRT scan effect. / Хроматическая аберрация привязана к переключателю CRT-эффекта и теперь отключается вместе со сканирующим CRT-эффектом.
- Updated CRT effect defaults so lens distortion and chromatic aberration keep their configured enabled intensities after scene and volume-profile refreshes. / Обновлены значения CRT-эффекта по умолчанию: искажение линзы и хроматическая аберрация сохраняют настроенную интенсивность после обновления сцены и профиля объема.
- Reduced Acoustic Visualizer texture size for better readability. / Уменьшен размер текстуры Визуализатора акустики для лучшей читаемости.
- Added interactive note inspection with cursor movement and zoom. / Добавлен интерактивный просмотр записки с перемещением по курсору и зумом.
